# Projects

这里存放 **Leo 侧项目管理状态**，用于自动同步到 GitHub `quantclaws/myclaws`。

原则：
- 这里只放项目卡、状态、验收记录、风险、决策链接。
- **不放业务代码**；业务代码仍留在各自 canonical repo。
- 运行数据、缓存、日志不进入这里。
- Leo 本地统一从 `~/workspace/projects/<project>/` 访问；该路径会映射到本仓库下的 `projects/`。

当前项目：
- `content-factory/` → canonical code repo: `quantclaws/content-factory`
